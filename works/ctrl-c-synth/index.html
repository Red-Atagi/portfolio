Ctrl + C Synth

2024

Medium: Web Browser

For this project, I was tasked with creating a musical instrument using a web browserâ€”without simply recreating an existing instrument. I drew inspiration from how I navigate and interact with my computer, specifically the familiar shortcuts "ctrl + c" and "ctrl + v" for copy and paste.

The instrument transforms these actions into part of its musical interface, turning routine computer gestures into creative expression. Additionally, the website allows users to explore microtones by moving their mouse across different sections of the page, making every pixel a unique pitch. The result is an experimental, web-based sound experience that blends everyday digital actions with playful musical exploration.
